•     Testing.
Some software programs offer the ability to run split tests. This means that a business can pilot a personalization effort with 1/2 of its customers and assess whether the results outperformed a non-personalization approach. Thus, this feature supports effective continuous improvement.

•     Auto-Traffic Allocation.
This is a special feature that can be built onto testing. Because A/B tests have an inherent cost (1/2 of users are sent the less effective route) some businesses are reluctant to perform them. Auto allocation means that the system running the test will automatically begin to send more traffic along the more successful route as the ‘winning’ test emerges.

•     Campaign Level Personalization.
This refers to being able to implement a set of choices across multiple categories to form a personalization campaign. Effectively, this means that when you have identified a system that is working, it is easy to automate this as a set of rules for other customers who visit your services digitally – across web, mobile, and email modalities.
