Planning the Test

Planning the Test

In this phase, your work is to:

  • Define what you are testing
  • Define how you will test it
  • Decide who you will need to test with
  • Decide what method of test you will implement
  • Select the metrics that will be collected during the test
  • Select a date(s) to run the test
  • Create a test script
  • Recruit participants for the test

Conducting Test

Conducting Test

In this phase, your work is to:

  • Prepare the test environment
  • Greet participants
  • Implement the test script for the interaction
  • Record metrics for the session

Analysis and Reporting

Analysis and Reporting

In this phase, your work is to:

  • Translate collected data into useful information
  • Draw conclusions from test results
  • Present data and insights in useful ways
